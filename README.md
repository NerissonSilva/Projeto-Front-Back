# 📚 Biblioteca Pessoal

Sistema web para gerenciar sua coleção pessoal de livros.

## 🚀 Tecnologias

- Next.js 15.3
- React 19.1
- React Query
- Axios
- Back4App (Backend)

## ⚙️ Configuração

1. Clone o repositório
2. Instale as dependências:
```bash
npm install
```

3. Configure as variáveis de ambiente:
   - Copie `.env.example` para `.env.local`
   - Adicione suas credenciais do Back4App

4. Execute o projeto:
```bash
npm run dev
```

5. Acesse: `http://localhost:3000`

## 📁 Estrutura

```
├── app/           # Páginas Next.js
├── api/           # Configuração API
├── components/    # Componentes React
└── styles/        # Estilos CSS
```

## 🔐 Funcionalidades

- Autenticação de usuários
- Adicionar/remover livros
- Marcar livros como lidos
- Recuperação de senha
